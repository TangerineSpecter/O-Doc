# O-Doc 项目文件结构

## 根目录结构

```
O-Doc/
├── .gitignore           # Git忽略文件配置
├── LICENSE              # 许可证文件
├── README.md            # 项目说明文档
├── eslint.config.js     # ESLint配置文件
├── index.html           # 应用入口HTML文件
├── package-lock.json    # npm依赖版本锁定文件
├── package.json         # npm项目配置文件
├── postcss.config.js    # PostCSS配置文件
├── public/              # 静态资源目录
│   └── vite.svg         # Vite默认SVG图标
├── src/                 # 源代码目录
├── tailwind.config.js   # Tailwind CSS配置文件
└── vite.config.js       # Vite构建工具配置文件
```

## src 目录结构

```
src/
├── App.css              # App组件样式文件
├── App.jsx              # 应用主组件
├── assets/              # 资源目录
│   └── react.svg        # React SVG图标
├── doc/                 # 文档目录
│   ├── frontend_doc/    # 前端文档
│   └── image.png        # 图片资源
├── index.css            # 全局样式文件
├── layout/              # 布局组件目录
│   └── Layout.jsx       # 主布局组件
├── main.jsx             # 应用入口文件
└── views/               # 页面视图组件目录
    ├── Article.jsx      # 文章页面组件
    ├── ArticleOutline.jsx # 文章大纲组件
    └── HomePage.jsx     # 主页组件
```

## 主要文件说明

### 布局与视图
- **Layout.jsx**: 实现了顶部导航栏和搜索模态框功能
- **HomePage.jsx**: 实现了文档集展示、筛选排序、无限滚动和新建文集功能
- **Article.jsx**: 文章详情页面组件
- **ArticleOutline.jsx**: 文章大纲组件，用于显示文章目录结构

### 配置文件
- **tailwind.config.js**: Tailwind CSS框架的配置文件
- **vite.config.js**: Vite构建工具的配置文件
- **eslint.config.js**: ESLint代码检查工具的配置文件
- **postcss.config.js**: PostCSS处理CSS的配置文件

### 入口文件
- **index.html**: HTML入口文件
- **main.jsx**: React应用入口文件，负责渲染根组件
- **App.jsx**: 应用主组件，负责组合Layout和视图组件
